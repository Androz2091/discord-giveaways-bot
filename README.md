## Install

* Clone repository:
```sh
git clone https://github.com/Androz2091/giveaways-bot
```

* Fill configuration (config.json):
```json
{
    "token": "Your discord bot token",
    "everyoneMention": false,
    "hostedBy": true,
    "guildId": "Your guild id"
}
```

* Install dependencies:
```sh
npm install
```

* Start:
```sh
node index.js
```
